# Pasta de Assets

Esta pasta deve conter as seguintes imagens:

## Imagens Necessárias

1. **logo-dark-tech.png** - Logo da Dark Tech
2. **banner-curso.jpg** - Banner principal do curso TechStarter
3. **foto-curso-1.jpg** - Foto do curso (opcional)
4. **foto-curso-2.jpg** - Foto do curso (opcional)

## Formato Recomendado

- **Logo**: PNG com fundo transparente
- **Banner**: JPG, resolução mínima 1200x600px
- **Fotos**: JPG, resolução mínima 800x600px

## Otimização

Para melhor performance, recomenda-se:
- Comprimir imagens antes de adicionar
- Usar formatos WebP quando possível
- Manter tamanhos de arquivo abaixo de 500KB cada
